# Έλεγχος Υλικού - Χτίζω Πρόταση

Χρησιμοποίησε αυτό το αρχείο για να μου σημειώσεις ποιες λέξεις/συνδυασμοί θέλεις να επιτρέπονται. Τα ids σε backticks είναι για να κάνω ακριβείς αλλαγές στον κώδικα.

## Δομές Πρότασης

Η κλιμάκωση ακολουθεί τη γλωσσική ανάπτυξη: πρώτα τα υποχρεωτικά συμπληρώματα του ρήματος (αντικείμενο), μετά οι προαιρετικοί προσδιορισμοί (τόπος, αιτία).

- Mode 2: ΠΟΙΟΣ + ΤΙ ΚΑΝΕΙ *(μόνο αμετάβατα ρήματα)*
- Mode 3: ΠΟΙΟΣ + ΤΙ ΚΑΝΕΙ + ΤΙ
- Mode 4: ΠΟΙΟΣ + ΤΙ ΚΑΝΕΙ + ΤΙ + ΠΟΥ
- Mode 5: ΠΟΙΟΣ + ΤΙ ΚΑΝΕΙ + ΤΙ + ΠΟΥ + ΓΙΑΤΙ

Η πλήρης χειροκίνητη λίστα των 164 εγκεκριμένων προτάσεων για το Mode 5 βρίσκεται στο `mode5-sentence-bank.md`. Τα Modes 3 και 4 παράγονται αυτόματα από τις προτάσεις του Mode 5 (κόβοντας αιτία και τόπο αντίστοιχα), ώστε κάθε μεταβατικό ρήμα να εμφανίζεται πάντα με το αντικείμενό του. Το Mode 2 έχει δική του επιμελημένη τράπεζα 50 προτάσεων με ρήματα που στέκονται φυσικά χωρίς αντικείμενο (κοιμάται, τρέχει, γελάει, χορεύει, πηδάει, τραγουδάει, κλαίει, περπατάει, κολυμπάει, πετάει), συν λίγα μεταβατικά σε αποδεκτή απόλυτη χρήση (τρώει, παίζει, διαβάζει, ζωγραφίζει, γράφει, μαγειρεύει, ψήνει). Το «πίνει» χωρίς αντικείμενο επιτρέπεται μόνο με ζώα, ώστε να αποφεύγεται η παρανόηση «πίνει αλκοόλ» που προκύπτει με ενήλικα πρόσωπα.

## Πώς Να Μου Γράψεις Τους Επιθυμητούς Συνδυασμούς

Μπορείς να γράψεις τους κανόνες σε αυτή τη μορφή:

```text
Ρήμα: τρώει / eat
Επιτρέπω αντικείμενα: μήλο, μπανάνα, ψωμί, τούρτα, καρότο, παγωτό, πίτσα
Επιτρέπω τόπους: σπίτι, κουζίνα, τραπέζι, εστιατόριο, πάρκο
Επιτρέπω αιτίες: γιατί πεινάει, γιατί είναι ώρα για φαγητό, γιατί είναι νόστιμο
Κόβω: τρώει + παγωτό + στο σχολείο, τρώει + πίτσα + στο πάτωμα
```

Ή ως πλήρεις προτάσεις:

```text
Το αγόρι τρώει το μήλο στο σπίτι γιατί πεινάει.
Το κορίτσι πίνει το νερό στο πάρκο γιατί διψάει.
Η μαμά διαβάζει το βιβλίο στο κρεβάτι γιατί θέλει να μάθει.
```
